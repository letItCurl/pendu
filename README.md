# Installation:
```
$ gem install pendu
```

# Usage:
```
$ pendu
// then the game start in your terminal...
// you can stop this with ctrl+c
```

# Uninstall:
```
gem uninstall pendu
// and if you are using rbenv:
rbenv rehash
```
